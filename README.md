# Setup

- Create a Virtual Environment

```
$ python -m venv venv
```

- Activate the Virtual Environment

```
MAC
$ source venv/bin/activate
(venv) $ _

PC
$ venv\Scripts\activate
(venv) $ _
```

- Install the packages

```
(venv) $ pipenv install
```

# Running Tests

```
(venv) $ pipenv run python -m pytest
```
